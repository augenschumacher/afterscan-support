2. Zweck der App

AfterScan PDF Scanner ermöglicht das Importieren vorhandener Fotos, das Bearbeiten von Dokumentkanten, das Einfügen von Signaturen und den Export als PDF oder Bilddatei.

3. Verarbeitung personenbezogener Daten

Die App verarbeitet die vom Nutzer ausgewählten Bilder und Dokumente lokal auf dem Gerät. Die Inhalte werden nicht an einen eigenen Server des Entwicklers übertragen.

4. Zugriff auf Fotos und Dateien

Die App kann Zugriff auf Fotos oder Dateien anfordern, damit Nutzer Dokumente importieren und bearbeiten können. Der Zugriff erfolgt nur, wenn der Nutzer ihn aktiv erlaubt.

5. Signaturen

Wenn Nutzer eine Signatur erstellen oder einfügen, wird diese zur Bearbeitung des jeweiligen Dokuments verwendet. Eine Übertragung an einen eigenen Server des Entwicklers findet nicht statt.

6. Weitergabe von Daten

Es erfolgt keine Weitergabe personenbezogener Daten an Dritte durch den Entwickler.

7. Analyse, Tracking und Werbung

Die App verwendet nach aktuellem Stand keine eigenen Analyse-, Tracking- oder Werbedienste.
